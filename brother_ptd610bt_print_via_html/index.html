Controls
Class summary
  • .label — base label; vertically centered; add one of .h-left, .h-center, .h-right for horizontal alignment.
  • .label.fold — two halves (1.5″ each for a 3″ label); optional printed fold line.
  • .half — a half in folded labels; apply .h-left/.h-center/.h-right per half.
  • .gap-5mm / .flag-gap — 5 mm spacer for flag/fold clearance (use .gap-5mm in flex/grid).
  • .fill — centers text inside the remaining space when using spacers.
